GET · REST · XML
선박운항정보

GET · REST · XML

GET · REST · XML

// Data Pipeline

데이터 처리 흐름

01
선박 목록 수집
선박입출항현황 목록으로 입항 선박 목록 및 계선장소 파악
02
선박 식별
IMO 번호·호출부호 매핑으로 선박 동일성 검증 및 확장 조회
03
화물 API 조회
선박별 벌크·컨테이너 반출입 API 호출, MRN 역매핑 재시도
04
부두별 집계
계선장소코드 기준으로 부두별 물동량 집계 및 통계집 대조 검증
// Purpose

활용 목적

📦
부두별 물동량 분석
부두별 벌크·컨테이너 연간 물동량을 집계하고 연도별 추이를 분석합니다.
🚢
선박·화물 연계
입항 선박 정보와 화물 데이터를 연계하여 선종별·항로별 물동량 패턴을 파악합니다.
📊
장기 시계열 구축
연도별 데이터베이스를 구축하여 항만 물류 트렌드를 분석합니다.
